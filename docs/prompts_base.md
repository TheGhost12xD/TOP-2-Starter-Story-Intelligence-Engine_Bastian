# Documento de Prompts Base Reutilizables

Este documento almacena los prompts base y estructurados que utilizaremos en los distintos módulos (Scraper, Clasificador, Perfil RPM, Motor de Soluciones) del proyecto Starter Story LATAM Analyzer.

---

## 📋 Plantilla Estándar de Prompt

Copia y pega esta plantilla cada vez que documentes un nuevo prompt:

### [Nombre o ID del Prompt]

**Objetivo:** 
> ¿Qué se espera lograr con este prompt? (ej. Extraer los datos de ingresos de un texto).

**Contexto:** 
> Información adicional que el modelo de lenguaje (LLM) necesita saber antes de procesar la solicitud (ej. "Eres un experto analista de negocios enfocado en startups latinoamericanas...").

**Instrucciones:**
1. [Paso 1...]
2. [Paso 2...]
3. [Restricciones o reglas importantes, ej. "No inventes información que no esté en el texto original"].

**Output Esperado:**
> Define el formato exacto en el que el LLM debe responder (ej. JSON estructurado, lista de viñetas, tabla Markdown).
